Frontend (Web)
==============

Conteúdo:
- `src/`: código-fonte (SPA ou páginas)
- `public/`: arquivos estáticos
- `package.json`: scripts e dependências

Scripts sugeridos (exemplo Vite + React):

```json
{
  "name": "frontend",
  "private": true,
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
```
